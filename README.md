# MKS Challenge Back-end

Projeto realizado para o desafio tecnico MKS

# Tecnologias usadas:

- Nestjs
- Typescript
- TypeOrm
- PostgreSQL

# Como rodar

- instale as dependencias
- copie o conteudo do arquivo .env.example para um arquivo chamado .env.local e altere as variaveis de ambiente
- npm run start:dev
